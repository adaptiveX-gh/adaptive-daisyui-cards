Shared Components Test

Test Instructions

  1. Open streaming-progressive.html in your browser
  2. Check that the "Shared Components" section appears with toggles and edit buttons
  3. Toggle "Show Headers" ON - verify headers appear on all cards when streaming starts
  4. Click "Edit Header" - verify modal opens with current values
  5. Change the header title and background color, click Save
  6. Verify header updates on all cards immediately
  7. Refresh the page - verify header settings persist from localStorage
  8. Repeat steps 3-7 for footers
  9. Open index.html in another tab
  10. Verify settings sync between both pages (same localStorage key)

Expected Behavior

  • Headers/footers toggle ON/OFF instantly on all cards
  • Edit buttons are disabled when headers/footers are OFF
  • Color picker shows selected color with highlighted border
  • Changes persist across page refreshes
  • Settings sync between index.html and streaming-progressive.html

localStorage Key

Check browser DevTools → Application → Local Storage
